## Changes Made

### 1. ✅ Created DateFormatUtils (NEW FILE)
**File**: [server/utils/dateFormatUtils.ts](server/utils/dateFormatUtils.ts)

Centralized all date parsing and comparison logic into a single utility class with methods:
- `parseUKDateToDate()` - Parse DD/MM/YYYY format
- `parseISODateToDate()` - Parse YYYY-MM-DD format
- `parseDate()` - Auto-detect format
- `toDateOnlyISO()` - Convert any format to ISO
- `isDateInPast()` - Check if date is past
- `isDateToday()` - Check if date is today
- `hasTimePassedOnDate()` - Check if time on today has passed
- `isSessionInPast()` - Comprehensive check combining date and time

**Benefits**:
- Single source of truth for date logic
- Eliminates 7+ instances of duplicate date parsing
- Easy to test and maintain
